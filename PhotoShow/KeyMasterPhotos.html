KeyMaster Web App

The KeyMaster web app was created with the intent for a user to be able to keep track of what PC games they own, what platform they purchased the game on, as well as any games that are on their wish list. A user is able to add games to their lists, including game title, genre, platform, as well as any associated game key number. A user can then check the Game List page to see all their games in one spot instead of having to go through their accounts on Steam, Epic Games, GOG, etc. in order to see what games they have.

Features

  • User ability to add, edit, and delete game enteries
  • Quick reference to entire list of games owned and on wish list

Tech

  • Python Programming Language
  • PyCharm IDE
  • Django Framework
  • MVC Architecture
  • CRUD Functionality
