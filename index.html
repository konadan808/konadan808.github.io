Theater Site Live Project

For this session, I worked as part of a team to create admin functions for a website that is owned by a local live theater. My portion of the project was to create and implement the production MVC using a code first Entity Framework Database. This allowed a user to create, edit, and delete entities that represented productions with data regarding a production name, playwright, run time, etc. I ensured that the layout of each CRUD page conformed to the established formatting, naming conventions, and layout per project parameters.
